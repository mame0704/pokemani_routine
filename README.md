# ポケマニルーティン💰

"ルーティン"×"お小遣い"アプリ

## サービス概要

- ポケマニルーティンは"ルーティン"×"お小遣い"アプリです。

「子供の自立を促しつつ親子でのコミュニケーションのきっかけ」を作るサービスです。

お子様に毎日「宿題は？」「リュック置きっぱなし😡」と注意することに疲れていませんか？
時に言いすぎて自己嫌悪に陥ったり、親子の衝突で疲弊してしまう...
そんな日々のストレスを減らし、楽しい親子時間を増やすためのアプリです。

## このサービスへの想い・作りたい理由

私は5人兄弟の長男で、末の弟(13歳)が宿題をせずに両親に叱られる場面をよく目にします。
仕事で疲れた後だと、親も私も感情的になってしまい、結果的に同じことを繰り返す負のループに...

「叱る→反発→また忘れる→また叱る」というサイクルを無くし、両親の負担を減らしたい。子供が自発的に行動できる仕組みを作りたい。
そして1番は **“家族が笑顔で過ごせる時間を増やしたい”** というのがこのアプリを作る一番の目的です。そして、このサービスがより多くの同じ悩みを抱えた家庭に届いて笑顔が増えることを願っています。

## ユーザー層について

対象としては見守ってもらう人と見守る人の2ユーザーが利用する形になっております。

- 見守ってもらう人👦：小〜中学生
- 見守る人👩：お子様がいるご両親

## サービスの利用イメージ

例）

### 👩 親の設定(初回のみ)

1. 子供用アカウントを作成
2. 日々のルーティンを登録(例:「宿題をする」10ポイント、「リュックを片付ける」5ポイント)
3. 1ポイント=○円の換算レートを設定

### 👦 子供の日々の使い方

1. 朝/夕方にアプリを開く
2. 今日のルーティンチェックリストを確認
3. 完了した項目にチェック✅
4. 親に「確認してね！」依頼

### 👩 親の承認作業

1. 実際にできているか確認👀

2. OKなら承認ボタン→ポイント付与💰
3. できていなければ「もう一度やってみよう」と返却

### 💡 ボーナス機能（本リリースで実装予定）

- お手伝い(食器洗い、掃除など)をしたら親が追加ポイントを付与

## ユーザーの獲得について

### 初期フェーズ(MVP)

- 自身の家族(両親、弟)に実際に使ってもらい、リアルなフィードバックを収集
- 使用感や改善点を週次でヒアリングし、素早くPDCAを回す
- 知人の小中学生の子供がいる家庭に直接アプローチ

### 成長フェーズ(本リリース後)

- X(Twitter)共有機能

## サービスの差別化ポイント・推しポイント

### コアコンセプト

「子供の自立」×「お小遣い」

### 差別化ポイント

1. お小遣いと連動したポイントシステム　ルーティン×お小遣いアプリ

- 単なるタスク管理ではなく、「お小遣い」という子供にとっても分かりやすいインセンティブを設定

2. 親子の関係性を前提とした設計

- 子供:実行する人 / 親:承認する人という明確な役割分担
- 教育的な要素を持つ
- 「承認」を通じて親子のコミュニケーションが生まれる仕組み

3. 「叱る」から「承認する」へ

- 従来:できていないことを指摘→感情的な衝突

　　本アプリ:できたことを承認→ポジティブなフィードバックループ→親の負担軽減

- 子供は「認めてもらえる喜び」を体験し、自己肯定感が向上

4.「子供の自立を促しつつ親子でのコミュニケーションのきっかけを作る」というビジョンを設定

- アプリ内やプロダクト画像で文言を記載することによって利用者が得られることやアプリ独自のイメージができるように可視化

## 機能候補

(MVP機能)

- ユーザー登録 / ログイン
- 親 / 子のロール分け
- ルーティン作成：ルーティン内容、実行頻度
- 子によるルーティン実行、チェック
- 親による承認
- アプリ内通知機能：チェック忘れ、ルーティン承認依頼、修正依頼

(本リリース機能)

- お小遣いボーナス機能：ポイント加算
- よくするルーティンのテンプレート機能
- ブラウザ通知機能：アプリ内通知と同様
- googleログイン機能
- パスワードリセット機能
- X共有機能
